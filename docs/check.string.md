### String `(check.string)`
#### `contains(str, sub)`
Check to see if `sub` is inside of `str`

#### `caps(str)`
Check to see if `str` is all caps.

#### `lowercase(str)`
Check to see if `str` is all lowercase.

#### `begins(str, sub)`
Check to see if `str` begins with `sub`.

#### `ends(str, sub)`
Check to see if `str` ends with `sub`.

#### `empty(str)`
Check to see if `str` is empty.
